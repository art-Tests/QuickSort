# 快速排序演算法

## 規則

1. 從陣列中取得一個基準點 (pivot)
1. 將整個陣列與基準點比較，區分成兩個區域，左側的區域都比基準點小；右側區域都比基準點大
1. 將左、右兩個區域的小陣列再次重複上述步驟
1. 將所有結果合併起來就是已排序過的結果
